## Google Docs

#### **Numbered list starting at 0**

1. Create a numbered list.
2. Place the text cursor _inside_ the numbered list.
3. Go to `Format` > `Bullets & numbering` > `List options` > `Restart numbering`.
4. Enter `0`.
5. Click `OK`.
